# MRI Segmentations

This directory contains manual claustrum segmentations from three in vivo 7-Tesla MP2RAGE MRI
datasets acquired at isotropic resolutions of 0.5 mm, 0.7 mm, and 1.0 mm (n=10 per dataset). Segmentations were performed manually in each subject's native space. 

---

## Label system

| Label | Structure |
| --- | --- |
| `1` | Left claustrum |
| `2` | Right claustrum |
