# MRI Data

This directory contains preprocessed, brain-masked MP2RAGE UNI images from two in vivo
7-Tesla MRI datasets (0.7 mm and 1.0 mm isotropic; n=10 per dataset) from Maastricht University, on which the manual
claustrum segmentations in `MRI/segmentations/` were performed.

The 0.5 mm isotropic dataset is not included here as it is publicly available as part of the
MICA-PNI dataset at [osf.io/mhq3f/overview](https://osf.io/mhq3f/overview)
([Cabalo et al., 2025](https://doi.org/10.1038/s41597-025-04863-7)).
